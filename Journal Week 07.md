# Week 07 - Learning Journal - PHP

## Learning Activities & Resources
This week I worked on learning the basics of PHP by creating a small multi-page website using a local server environment (Local by Flywheel).  

I created multiple PHP files including a header, footer, and two content pages, and used the `include` function to reuse common elements across pages.  

I practiced core PHP concepts such as:
- Using `echo` to display HTML content
- Writing `if/else` statements for decision making
- Using loops (`for`, `while`, and `foreach`) for repetition
- Creating functions with parameters to generate reusable content
- Working with arrays to store and display structured data  

This helped me understand how PHP integrates with HTML to create dynamic web pages. The website was a simple movie recommandation with a login page, A time checker to see if the user has enough time to watch a shorter film or and longer film.

## Estimated Hours
1–2 hours

## Content Insights
I learned how PHP runs on the server and generates HTML that is sent to the browser.  

Using `include` made the site much easier to manage, as I could reuse the same header and footer across multiple pages instead of rewriting code.  

Loops and arrays were especially useful for displaying repeated content like lists of items, while functions helped reduce duplication and made the code more organised.  

I also gained a better understanding of how logic (like `if/else`) can control what content is shown to the user.

## Career
