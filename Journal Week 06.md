# Week 06 - learning Journal - Group Setup
## Learning Activities & Resources
Group Setup involved configuring and integrating key teamwork tools including GitHub, Slack, LinkedIn, and Trello.  
Joined the GitHub repository through the classroom link, made an initial test commit and added slack intergation, and confirmed communication through a Slack group message.  
Set up and organised a Trello board to manage tasks and workflows, and completed the group agreement outlining roles, availibity, and responsibilities.  
## Estimated Hours
1 Hour
## Content Insights
Learned how different collaboration tools work together in a development environment. GitHub was used for version control and tracking changes, Slack for communication, Trello for task management, and LinkedIn for professional presence.  

## Career  
These tools are widely used in the tech industry, particularly in software development and IT project management. Learning how to use GitHub, Slack, and Trello reflects real-world workflows and prepares for professional team environments.

## Employability  
Developing experience with industry-standard tools enhances employability by demonstrating the ability to collaborate effectively in a team, manage tasks, and use version control systems. These are highly valued skills by employers.

## Learning insights  
This activity highlighted the importance of clear communication and organisation when working in a team. Setting up tools early makes collaboration smoother and reduces confusion later in the project. It also reinforced the importance of documenting contributions and maintaining consistent workflow practices.
